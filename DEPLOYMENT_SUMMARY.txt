5. API接口
================================================================================

信号监控 API (端口 5003):
  GET /api/signal-monitor/latest   - 获取最新信号
  GET /api/signal-monitor/history  - 查询历史数据
  GET /api/signal-monitor/stats    - 获取统计信息
  GET /api/signal-monitor/test     - 测试API连接

恐慌指标 API (端口 5002):
  GET /api/panic-wash/latest       - 获取最新数据
  GET /api/panic-wash/history      - 获取历史数据
  GET /api/panic-wash/query        - 查询指定时间范围
  GET /api/panic-wash/stats        - 获取统计信息

主服务器代理 (端口 5001):
  所有API请求通过主服务器代理访问
