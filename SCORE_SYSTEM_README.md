## ✨ 核心功能

### 📊 多时间段统计

系统支持 **6个时间段** 的得分分析：

```
✅ 3分钟   - 超短线参考
✅ 1小时   - 短线交易
✅ 3小时   - 日内波段
✅ 6小时   - 半日趋势
✅ 12小时  - 日内整体
✅ 24小时  - 日线趋势
```

### 📈 统计指标

每个时间段提供以下指标：

| 指标 | 说明 | 示例 |
|------|------|------|
| **平均做多得分** | 所有币种做多得分的平均值 | 49.13 |
| **平均做空得分** | 所有币种做空得分的平均值 | 50.99 |
| **平均差值** | 做多得分 - 做空得分 | -1.87 |
| **趋势判断** | 基于差值的市场倾向 | 📉 看空 |

### 💰 币种支持

系统支持 **19种主流加密货币**：

```
BTC, ETH, BNB, SOL, DOT, LINK, ADA, FIL, DOGE,
UNI, TAO, CFX, BCH, XLM, HBAR, ETC, AVAX, MATIC, OKB
```

可扩展至更多币种，支持多数据源整合。

## 🔌 API 使用

### 1. 获取统计数据

```bash
GET /api/score/statistics
```

**响应示例：**
```json
{
    "update_time": "2025-12-03 15:19:47",
    "statistics": [
        {
            "time_range": "3m",
            "avg_long_score": 49.13,
            "avg_short_score": 50.99,
            "avg_diff": -1.87,
            "coin_count": 12,
            "trend": "📉 看空"
        }
    ]
}
```

### 2. 获取币种详细数据

```bash
GET /api/score/coins?hours=24
```

**响应示例：**
```json
{
    "BTC-USDT-SWAP": {
        "3m": {
            "long_score": 52.34,
            "short_score": 48.76,
            "diff": 3.58,
            "update_time": "2025-12-03 15:19:47"
        }
    }
}
```

### 3. 手动刷新数据

```bash
GET /api/score/refresh
```

## 📈 数据解读

### 平均差值说明

| 差值范围 | 市场情绪 | 建议 |
|----------|---------|------|
| **> +2.0** | 🟢 强势看多 | 考虑做多 |
| **+0.5 ~ +2.0** | 🟡 温和看多 | 谨慎做多 |
| **-0.5 ~ +0.5** | ⚪ 中性平衡 | 观望为主 |
| **-2.0 ~ -0.5** | 🟡 温和看空 | 谨慎做空 |
| **< -2.0** | 🔴 强势看空 | 考虑做空 |

### 时间段选择

- **3分钟/1小时：** 适合短线交易，波动大
- **3小时/6小时：** 适合日内波段，较稳定
- **12小时/24小时：** 适合趋势交易，最稳定
