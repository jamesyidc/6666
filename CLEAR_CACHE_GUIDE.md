# 🔄 如何清除浏览器缓存查看完整29个币种

## ⚠️ 问题原因

您的浏览器缓存了旧版本的页面（只有10个币种），需要强制刷新才能看到新的29个币种。

## 💡 解决方法

### 方法1: 强制刷新（推荐，最快）

访问页面后，按以下快捷键强制刷新：

- **Windows / Linux**: `Ctrl + F5` 或 `Ctrl + Shift + R`
- **Mac**: `Cmd + Shift + R`

### 方法2: 清除浏览器缓存

#### Chrome / Edge
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 选择"时间范围"为"过去1小时"
3. 勾选"缓存的图片和文件"
4. 点击"清除数据"
5. 重新访问页面

#### Firefox
1. 按 `Ctrl + Shift + Delete` (Windows) 或 `Cmd + Shift + Delete` (Mac)
2. 选择"时间范围"为"最近1小时"
3. 勾选"缓存"
4. 点击"清除"
5. 重新访问页面

#### Safari (Mac)
1. 按 `Cmd + Option + E` 清空缓存
2. 或者：Safari菜单 > 偏好设置 > 隐私 > 管理网站数据 > 移除全部
3. 重新访问页面

### 方法3: 无痕模式/隐私模式

1. 打开浏览器的无痕/隐私模式
2. 访问页面
3. 这样可以避免使用缓存

**快捷键**:
- Chrome/Edge: `Ctrl + Shift + N` (Windows) 或 `Cmd + Shift + N` (Mac)
- Firefox: `Ctrl + Shift + P` (Windows) 或 `Cmd + Shift + P` (Mac)
- Safari: `Cmd + Shift + N`

### 方法4: 开发者工具强制刷新

1. 按 `F12` 打开开发者工具
2. 右键点击浏览器的刷新按钮
3. 选择"清空缓存并硬性重新加载"
